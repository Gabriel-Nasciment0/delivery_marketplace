# 🧠 3. Regras de Negócio

* [ ] Produto repetido soma quantidade
* [ ] Quantidade mínima = 1
* [ ] Carrinho vazio não finaliza pedido
* [ ] Total atualizado automaticamente
* [ ] Pedido deve conter:

  * produtos
  * total
  * forma de pagamento
  * dados do cliente
* [ ] PIX:

  * status = aguardando comprovante
* [ ] Pagamento local:

  * status = pagamento na entrega

# 🧭 5. Fluxo do Usuário

## 🔹 Cliente

```id="userflow"
Usuário entra →
visualiza produtos →
adiciona ao carrinho →
abre carrinho →
confirma itens →
vai para checkout →
preenche dados →
escolhe pagamento →
finaliza pedido
```

---

## 🔹 Admin

```id="adminflow"
Admin entra →
visualiza produtos →
gerencia produtos →
visualiza pedidos
```
