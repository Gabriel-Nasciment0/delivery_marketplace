# 🧩 Requisitos do Projeto — Delivery Multicategoria

## 📌 1. Requisitos Funcionais

### 🔹 Cliente

* [ ] Visualizar a página inicial com categorias de comida
* [ ] Visualizar lista de produtos por categoria
* [ ] Filtrar produtos por tipo de comida
* [ ] Visualizar nome, preço, descrição e imagem dos produtos
* [ ] Adicionar produtos ao carrinho
* [ ] Remover produtos do carrinho
* [ ] Alterar quantidade dos produtos no carrinho
* [ ] Visualizar total do pedido
* [ ] Finalizar pedido
* [ ] Escolher forma de pagamento
* [ ] Informar dados do cliente
* [ ] Enviar pedido via WhatsApp
* [ ] Navegar entre páginas

### 🔹 Carrinho

* [ ] Manter os itens durante a navegação entre páginas
* [ ] Somar quantidades corretamente quando o mesmo produto for adicionado novamente
* [ ] Calcular o total automaticamente
* [ ] Exibir carrinho vazio quando não houver itens

### 🔹 Checkout

* [ ] Exibir formulário controlado
* [ ] Validar campos obrigatórios
* [ ] Selecionar forma de pagamento:

  * PIX
  * Dinheiro
  * Cartão
* [ ] Exibir instruções de pagamento de acordo com a opção escolhida
* [ ] Confirmar pedido antes do envio

### 🔹 Admin

* [ ] Visualizar produtos cadastrados
* [ ] Visualizar pedidos realizados
* [ ] Adicionar produto
* [ ] Editar produto
* [ ] Remover produto
* [ ] Organizar produtos por categoria

## 🧠 3. Regras de Negócio

* [ ] Produto repetido soma quantidade no carrinho
* [ ] Quantidade mínima de item no carrinho é 1
* [ ] Carrinho vazio não pode ser finalizado
* [ ] Total deve ser atualizado automaticamente
* [ ] Pedido deve conter:

  * produtos
  * total
  * forma de pagamento
  * dados do cliente
* [ ] Se a forma de pagamento for PIX, o sistema deve mostrar instruções para envio do comprovante
* [ ] Se a forma de pagamento for no local, o sistema deve registrar isso no pedido
* [ ] Os produtos devem ser organizados por categoria
* [ ] O sistema deve permitir visualização separada dos itens por categoria

## 🧭 5. Fluxo do Usuário

### 🔹 Cliente

```txt
Usuário entra →
visualiza categorias →
seleciona uma categoria →
visualiza produtos →
adiciona ao carrinho →
abre carrinho →

preenche dados →
escolhe pagamento →
finaliza pedido →
recebe confirmação via WhatsApp
```

### 🔹 Admin

```txt
Admin entra →
visualiza produtos →
cadastra/edita/remove produtos →
visualiza pedidos realizados
```
