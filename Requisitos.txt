## 🔹 Objetivo

Desenvolver uma aplicação web de delivery multicategoria para pedidos online.

O sistema permitirá que clientes:

* visualizem produtos
* filtrem categorias
* adicionem itens ao carrinho
* finalizem pedidos

Tudo dentro de um único delivery que vende diferentes tipos de comida:

* hambúrguer
* pizza
* comida japonesa
* bebidas
* sobremesas

O objetivo principal é criar um sistema moderno, organizado e escalável para prática de desenvolvimento Front-End com React.

# 📌 2. Regras de Negócio

• Produto repetido soma quantidade
• Quantidade mínima = 1
• Carrinho vazio não pode finalizar pedido
• Total atualiza automaticamente
• Pedido deve conter:

* produtos
* total
* forma de pagamento
* dados do cliente

• PIX:

* pagamento pendente até comprovante

• Pagamento presencial:

* status = pagar na entrega/retirada

• Produtos devem pertencer a uma categoria
